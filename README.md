# Todo List Speedy ✅

A web-based task management application

## Project Overview

Todo List Speedy helps users organize daily tasks through a simple and fast interface.

This project was built to practice:

- JavaScript programming
- Frontend development
- DOM manipulation
- Event handling
- User interface design

## Features

✅ Add tasks  
✅ Delete tasks  
✅ Mark tasks as completed  
✅ Simple task management interface  
✅ Fast and easy workflow  

## Technologies Used

- JavaScript
- HTML
- CSS

## Installation

Clone repository:

```bash
git clone https://github.com/andeskdeng/todo-list-speedy.git
cd todo-list-speedy
```

Open:

```bash
index.html
```

Or launch with:

```bash
start index.html
```
